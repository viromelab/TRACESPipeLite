# TRACESPipeLite

Lite version of TRACESPipe [viral metagenomics only]

## Installation ##

```
git clone https://github.com/viromelab/TRACESPipeLite.git
cd TRACESPipeLite/src/
lzma -d VDB.fa.lzma
chmod +x *.sh
./TRACESPipeLite.sh --install
```
